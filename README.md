# Estudo de Caso - Regressão Linear em base de acompanhamento de diabetes

Este estudo utiliza como base dados fornecidos na documentação da biblioteca SciKit-Learn. Seu objetivo é fornecer uma
análise clara e objetiva sobre o acompanhamento da evolução de casos de diabetes em pacientes variados, não identificados, com diversidade
de sexos e condições físicas.
Foram utilizados modelos de regressão linear e métodos de análise bastante conhecidos com intenção de estudo e aperfeiçoamento de minhas praticas 
como cientista de dados.

Link da documentação da base de dados: [8.1.2. Diabetes dataset](https://scikit-learn.org/stable/datasets/toy_dataset.html#diabetes-dataset)

Para visualizar mais repositórios ou estabalecer contato, estou a disposição no GitHub como [Pedro Caovila](https://github.com/pedrocaovila) ou
